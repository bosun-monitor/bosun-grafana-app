Enable Query Helper This enables the the OpenTSDB query helper when using the datasource. If you are using Bosun with OpenTSDB and are indexing your OpenTSDB data to Bosun, you can use this to build q() functions based on the metadata Bosun has.
Enable pre-release features This enables new features that depend on unreleased versions of Bosun. If you're running of a recent build, you can enable those new features now.
OpenTSDB URL Required for metric autocompletion
Annotate URL Link to root of bosun UI for links to edit annotations. Usually the same as the datasource. To enable annotations in Bosun, you need elastic and something like the following in Bosun's configuration: annotateElasticHosts=http://ny-lselastic01.ds.stackexchange.com:9200,http://ny-lselastic02.ds.stackexchange.com:9200
Auth Token If auth is enabled, this token will be used to for requests made to bosun. Note that the storage of the token is currently not secure at this time.
